# [SSU:Tructure](https://vvsos1.github.io/ssutructure/)

**SSU:Tructure** 는 **자료구조**와 **알고리즘**의 **동작 과정**을 웹으로 **시뮬레이션** 할 수 있는 프로젝트입니다.

## Getting Started

### Prerequisites
```
Linux
Windows WSL
Windows Git Bash
```
```
Node js & NPM
```

### Installing
with npm : 
```
npm install
```

### Build
To Build : 
```
git clone https://github.com/vvsos1/ssutructure
cd ssutructure
bash build.sh
cd output
```

### Local Server Start
After Build: 
```
bash start-server.sh
```
